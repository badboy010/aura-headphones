## 
Aura Headphones
Sounds that fits your life.
Aura Headphones is a modern, full-stack e-commerce platform built for the Internet Computer
(ICP). It combines a high-performance React frontend with a robust Motoko backend to deliver a
decentralized shopping experience.
Overview
Aura Headphones showcases a premium brand identity through a high-contrast Neon Green and
Black visual theme. The application provides a seamless flow from product discovery to checkout,
leveraging the speed and security of blockchain technology.
Features
Product Catalog – A curated collection of premium headphone styles.
Persistent Shopping Cart – Add, remove, and manage items in real-time.
Integrated Checkout Flow – A streamlined, user-friendly purchase process.
Customer Social Proof – Integrated testimonials from verified users.
Responsive UI – Fully optimized for mobile, tablet, and desktop viewing.
Modern Branding – Distinctive "Aura" visual identity using a bold neon palette.
Tech Stack
Layer Technology
Frontend React, TypeScript, Tailwind CSS
Backend Motoko (Internet Computer)
Hosting Internet Computer (ICP)

Installation
1. Clone the repository:
git clone https://github.com/badboy010/aura-headphones.git
cd aura-headphones
2. Install dependencies:
npm install
3. Local Development:
dfx start --background
dfx deploy
Alternatively, to run just the frontend development server:
npm start
